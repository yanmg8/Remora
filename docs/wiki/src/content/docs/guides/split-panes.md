title: 分栏布局
description: 了解如何使用多标签和分栏布局提升工作效率。
---

Remora 支持灵活的多标签和分栏布局，让您同时管理多个会话。

## 标签页操作

### 基本操作

Remora 使用标准 macOS 标签页操作：

- 使用 **Cmd+T** 新建标签（通过菜单或快捷键）
- 使用 **Cmd+W** 关闭标签

### 标签右键菜单

右键点击标签可进行：

- 关闭标签
- 关闭右侧标签
- 复制标签内容到新标签

## 分栏操作

分栏让您在一个标签内并排查看多个终端。

### 创建分栏

1. 右键点击标签
2. 选择 **Split Horizontal**（水平分栏）或 **Split Vertical**（垂直分栏）

注意：每个标签最多支持 2 个分栏。

### 调整分栏大小

拖动分栏分隔线可调整各分栏大小。

## 布局组合

### 常用布局

- **多标签模式**：每个标签一个会话，适合管理多个连接
- **单标签多分栏**：一个标签内两个分栏，适合对比或监控
- **混合模式**：结合标签和分栏，最大化工作效率

### 布局保存

Remora 会自动保存工作区布局，下次打开时恢复。
