## 应用快捷键

| 快捷键 | 说明 |
|--------|------|
| `⌘,` | 打开设置 |
| `⌘B` | 显示/隐藏侧边栏 |
| `⌘⇧N` | 新建 SSH 连接 |
| `⌘I` | 导入连接 |
| `⌘E` | 导出连接 |

## 终端快捷键

Remora 终端支持标准编辑快捷键：

- `⌘C` - 复制
- `⌘V` - 粘贴
- `⌘A` - 全选
- `⌘F` - 搜索

## 文件管理器快捷键

| 快捷键 | 说明 |
|--------|------|
| `Enter` | 打开文件/目录 |
| `Backspace` | 返回上级目录 |
| `Space` | 预览文件 |
| `Delete` | 删除选中 |
