# SFTP 参考

## 文件操作

Remora 提供完整的 SFTP 文件管理功能：

- **浏览**：在本地和远程目录之间导航
- **上传**：拖拽文件到目标目录
- **下载**：拖拽文件到本地目标
- **新建**：创建文件和目录
- **重命名**：修改文件/目录名称
- **删除**：删除文件或目录
- **复制/粘贴**：复制和粘贴文件

## 传输队列

所有传输操作都会进入队列，您可以：

- 查看传输进度
- 暂停/恢复传输
- 取消传输
- 重试失败的传输

## 传输设置

| 选项 | 说明 |
|------|------|
| 并发传输数 | 同时进行的传输任务数 |
| 保留时间戳 | 上传/下载时保留文件时间 |
| 覆盖策略 | 询问/覆盖/跳过 |
