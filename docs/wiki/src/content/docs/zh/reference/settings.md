# 设置参考

## 常规设置

### 外观

| 设置 | 说明 |
|------|------|
| 主题 | 亮色/暗色/跟随系统 |
| 缩放 | 界面缩放比例 |

### 语言

选择界面显示语言：
- English
- 简体中文

## 连接设置

### 默认连接参数

- 默认端口：22
- 连接超时：30 秒
- 保持连接间隔：60 秒

### 凭据存储

选择凭据存储方式：
- **仅 Keychain** (推荐)：安全存储在 macOS Keychain
- **记住密码**：短期内存储

## 终端设置

- 默认终端类型
- 启动 shell 命令
- 环境变量

## 高级设置

- 调试日志
- 网络诊断
- 重置所有设置
