# SSH 参考

## 连接参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 主机 | SSH 服务器地址 | - |
| 端口 | SSH 端口号 | 22 |
| 用户名 | SSH 用户名 | - |
| 认证方式 | 密码/密钥 | 密码 |

## 认证方式

### 密码认证

输入密码进行连接。密码会安全存储在 macOS Keychain 中。

### SSH 密钥认证

1. 点击 **选择密钥**
2. 选择私钥文件 (通常在 `~/.ssh/` 目录)
3. 如有 passphrase，输入密钥密码

## SSH 选项

- **保持连接**：定期发送保活包防止连接断开
- **压缩**：启用 zlib 压缩
- **转发代理**：配置 SSH 代理跳转
