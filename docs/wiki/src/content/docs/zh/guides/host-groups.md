# 主机分组

Remora 允许您将 SSH 主机分组，以便更好地组织连接。

## 创建主机分组

1. 点击侧边栏底部的 **+** 按钮
2. 选择 **新建分组**
3. 输入分组名称
4. 选择分组图标和颜色

## 添加主机到分组

1. 在主机列表中选择一个主机
2. 拖拽到目标分组
3. 或右键点击主机 → **移动到** → 选择分组

## 分组管理

- 重命名分组：右键点击分组 → **重命名**
- 删除分组：右键点击分组 → **删除**（主机不会被删除）
- 排序：拖拽调整分组顺序
