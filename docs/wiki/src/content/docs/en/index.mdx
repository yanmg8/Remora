## Features

<CardGrid stagger>
	<Card title="Fantastic" icon="star">
		Local-first SSH + SFTP workspace with ANSI/VT support for modern TUIs, xterm-style selection, quick commands/paths, and drag-and-drop transfers.
	</Card>
	<Card title="Beautiful" icon="laptop">
		Native macOS UI with clean split layout, light/dark/system themes, and distraction-free terminal focus.
	</Card>
	<Card title="Fast" icon="rocket">
		Swift 6 native architecture with a SwiftTerm-based terminal stack, integrated natively for practical TUI and scroll-heavy workflows.
	</Card>
	<Card title="Secure" icon="shield">
		Local-first credential strategy with config and saved passwords stored in local JSON files under `~/.config/remora`, plus SSH host key verification via StrictHostKeyChecking=ask.
	</Card>
	<Card title="Simple" icon="setting">
		Lightweight app with 99% Swift-native stack, keyboard-driven workflows, and practical defaults that work out of the box.
	</Card>
</CardGrid>

## What You Can Do Today

- Run local shell and SSH sessions with multi-tab/pane workspace
- Manage hosts with groups, search, favorites, and quick connect
- Use SFTP file manager for create/rename/move/delete/copy/paste/upload/download
- Drag files onto directories with visual upload target hints
- Get immediate operation feedback via toasts and retry failed transfers
