Remora supports flexible tab and split pane layouts, allowing you to manage multiple sessions simultaneously.

## Tab Operations

### Basic Operations

Remora uses standard macOS tab operations:

- Use **Cmd+T** to create a new tab (via menu or shortcut)
- Use **Cmd+W** to close a tab

### Tab Context Menu

Right-click a tab to:

- Close tab
- Close tabs to the right
- Duplicate tab content to new tab

## Split Pane Operations

Split panes let you view multiple terminals side by side within a single tab.

### Creating Splits

1. Right-click on a tab
2. Select **Split Horizontal** or **Split Vertical**

Note: Each tab supports a maximum of 2 panes.

### Resizing Splits

Drag the split divider to resize panes.

## Layout Combinations

### Common Layouts

- **Multi-tab mode**: One session per tab, great for managing multiple connections
- **Single tab, multiple panes**: Side-by-side terminals in one tab, perfect for monitoring
- **Mixed mode**: Combine tabs and panes for maximum efficiency

### Layout Persistence

Remora automatically saves your workspace layout and restores it on next launch.
