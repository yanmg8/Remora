Remora provides complete file transfer management.

## Transfer Queue

All SFTP transfers are managed in a unified queue.

### Viewing the Queue

Click the **Transfers** button in the sidebar to open the transfer panel.

### Queue Status

| Status | Description |
|--------|-------------|
| Pending | Waiting to transfer |
| Transferring | Currently transferring |
| Completed | Transfer successful |
| Failed | Transfer failed |

## Transfer Operations

### Cancel Transfer

Click cancel to terminate the transfer.

### Retry Failed Transfers

Click retry to retransfer failed files.

## Transfer Settings

Configure transfer options in **Settings > File Manager**.
