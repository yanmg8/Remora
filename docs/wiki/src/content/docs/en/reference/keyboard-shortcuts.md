This page lists keyboard shortcuts available in Remora.

## Application Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+,` | Open Settings |
| `Cmd+B` | Toggle sidebar |
| `Cmd+Shift+N` | New SSH Connection |
| `Cmd+I` | Import Connections |
| `Cmd+E` | Export Connections |

## Terminal Shortcuts

Remora terminal supports standard edit shortcuts:

- `Cmd+C` - Copy
- `Cmd+V` - Paste
- `Cmd+A` - Select All
- `Cmd+F` - Search

## File Manager Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Open file/folder |
| `Backspace` | Go to parent folder |
| `Space` | Quick Look |
| `Delete` | Delete selected |
