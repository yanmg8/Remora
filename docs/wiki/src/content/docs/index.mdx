## 你现在就可以做的事

- 在同一工作区里运行本地 Shell 与 SSH 会话（多标签/分栏）
- 管理主机分组、搜索、收藏，并使用快速连接
- 通过 SFTP 文件管理器执行新建，重命名，移动、删除、复制/粘贴、上传/下载
- 拖拽上传到目录或当前路径，带目标高亮与提示
- 获取即时操作反馈（toast）并重试失败传输
