

# [自荐] 做了个 macOS 原生 SSH + SFTP 工具 Remora，想听听大家最想要什么功能


最近在做一个 macOS 上的 SSH + SFTP 工具，叫 Remora。

为什么又造新轮子？必然是因为一直没在 macOS 上找到心怡的SSH管理工具。

之前也用过 WindTerm、ElecTerm、FinalShell 这些，很多都不是原生的。要么丑，要么不太流畅。

加上我自己也不太喜欢 web 套壳的应用，哪怕是 Rust 套壳，还是会更喜欢原生的东西。一直没找到合适的，就干脆自己做了。

我现在主要想解决几个问题：

- SSH 和 SFTP 很多时候是分开的，连上机器之后传文件还得切来切去

- 跑 vim、htop、k9s 这类 TUI 应用的时候，很多工具总会有些地方不顺

- 日志一长、输出一多，滚动和渲染体验就容易拉胯

- macOS 上一直缺一个我自己觉得顺手的原生方案

所以 Remora 现在的方向也比较明确：

- SSH、SFTP、文件传输放到一个工作区里

- 先把 TUI 兼容和大输出场景下的性能做好

- 整体尽量保持 macOS 原生应用的感觉

目前已经在做的包括：

- SwiftUI ，纯原生UI，也有自己的终端引擎

- 针对 vim、top/htop、opencode、claude 等应用做兼容

- 大输出、长日志场景下尽量保持顺滑

- SSH 和 SFTP 放在同一个界面里处理
  
- 当然还有很多交互细节的优化

后面准备补这些：

- 跳板机 / 多级连接

- 导入 ~/.ssh/config

- 更完整的密钥管理

- 同步 / 共享连接配置

- AI 支持，这块已经在做，先从服务器辅助操作、命令解释、日志理解这些方向开始
